Mooo Air Gapped Wallet

Open Source PSBT (Partially Signed Bitcoin Transactions) signer for Bitcoin.

Main futures :

  • Electrum compatible
  • Multiple accounts creation
  • Generate Native Segwit accounts
  • Generate Segwit Compatible accounts
  • QR codes to fully manage private keys in an air gapped fashion

Use at your own risk.

Check the source code, report issues and contribute to this project via Github
